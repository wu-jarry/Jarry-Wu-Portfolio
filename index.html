Portfolio website

I made the room and fish animation on Blender, and all the website controls, scrolling animations, and lighting were done through gsap and three.js. Also that smooth scrolling effect you're feeling is thanks to a tool called Lenis Scroll. It was definitely a learning experience setting thatup but it is very rewarding to have implemented so I hope you like it!
